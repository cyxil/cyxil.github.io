如果你好奇试了试，会发现博客打不开了。

别急，分类标签我们之后再配置，保留默认的title和date就可以了。

在博客里写上一些正文，清空缓存(hexo clean)、生成(hexo g)、提交(hexo d)到Github

打开我们的博客，多次刷新后，就可以看到我们的新文章了。

hexo除了基本的md语法，还有一套专属的标签，这里不多介绍了，大家可以到官方文档查看，Hexo文档

Hexo主题

Hexo有很多漂亮的主题 Hexo主题，可以根据自己喜好选择不同的主题。

下载主题

选择一个主题，进入对应的仓库，在下面的说明里基本都会有安装教程。

主题配置可能有差异，但下载都是一样的，把仓库clone到themes目录即可

比如我用的是Next主题，需要在根目录运行clone命令

启用主题

接着，把_config.yml 文件中的theme字段的值改为Next(文件夹名)，本地运行一下，会发现主题已经成功启用了。
